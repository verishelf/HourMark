# HourMark Website

Marketing site for the HourMark luxury watch marketplace iOS app. Built with Next.js 16, TypeScript, Tailwind CSS, and Framer Motion.

## Development

```bash
cd website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy to [Vercel](https://vercel.com). The repo includes a root `vercel.json` that builds only the `website/` app, so Vercel won't try to install the Expo mobile dependencies.

Alternatively, set the Vercel project **Root Directory** to `website`.

## Design

Matches the mobile app aesthetic:

- Background `#000000`
- Card `#0A0A0A`
- Border `#1A1A1A`
- Editorial typography and minimal luxury layout
